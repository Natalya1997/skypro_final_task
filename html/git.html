Git

  • git status - просмотр списка измененный файлов.
  • git checkout - сменить ветку.
  • git init -создание нового репозитория Git в пустой директории или превращения существующей директории в новый репозиторий Git.
  • git branch - создание ветки.
  • git commit - сохрание изменений в Git.
